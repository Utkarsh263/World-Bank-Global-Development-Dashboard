
# 🌍 World Bank Global Development Dashboard

An interactive **Power BI dashboard** built using **World Bank Open Data** to analyze global development indicators across **Economy, Human Development, Population, Technology, and Environment**. The project leverages **Power Query**, **DAX**, and interactive visualizations to provide actionable insights.

## 🚀 Features

- 📊 Multi-page interactive dashboard
- 📈 KPI Cards with dynamic DAX measures
- 🌍 Geographic Analysis using Maps
- 📉 Trend Analysis & Forecasting Visuals
- 🔍 Key Influencers & Decomposition Tree
- 📌 Ribbon, Waterfall, Funnel & Scatter Charts
- 📋 Matrix with Conditional Formatting
- 🎯 Dynamic slicers and cross-filtering

---

## 🛠 Tech Stack

- **Power BI Desktop**
- **Power Query**
